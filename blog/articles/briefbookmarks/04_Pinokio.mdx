# Brief Bookmarks - Pinokio

Nein, ich erzähle jetzt nichts über den Disney Klassiker über die Holzpuppe (Pinokio, nicht Pinocchio!) die ein Junge sein will, sondern über ein [kleines Tool](https://pinokio.computer/). Das Tool ist eigentlich nichts anderes als eine github Sammlung von AI tools und automatisiert die Installationen, Updates und Start-Up der Tools und nutzt eigentlich ohnehin nur standard tools von python: Virtuelle Umgebungen. Dadurch hat man voneinander abgekapselte Tools die sich gegenseitig nicht in die Quere kommen UND es ist ein wenig "kuratiert".
