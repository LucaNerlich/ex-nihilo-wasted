### Sourcecode

Nachfolgend der Code den ich auf den Puck geladen habe. Da ich keine Ahnung von Hardware, bzw Bluetoothgeraete Programmierung habe, musste ich mir das meiste dieses kurzen Scriptes zusammen-googlen, sowie ueber das Espruino Forum erfragen.

Der Puck kann bequem ueber die [Browser IDE](https://www.espruino.com/ide) bespielt werden.
Am einfachsten nutzt man dafuer Chrome und verbindet den Puck direkt ueber Bluetooth mit Chrome.

```javascript
// http://www.espruino.com/BLE+Keyboard
var kb = require("ble_hid_keyboard");
NRF.setServices(undefined, { hid: kb.report });

// https://forum.espruino.com/conversations/390668/#comment17145677
NRF.setAdvertising([
  2, 1, 6,  // standard Bluetooth
  8, 9, 80, 117, 99, 107, 46, 106, 115, // 'Puck.js' name
  3, 3, 0x12, 0x18, // HID Service
  3, 0x19, 0xc1, 0x03 // Appearance: Keyboard https://btprodspecificationrefs.blob.core.windows.net/assigned-numbers/Assigned%20Number%20Types/Assigned_Numbers.pdf
]);

function btnPressed() {
  kb.tap(kb.KEY.B, kb.MODIFY.LEFT_GUI | kb.MODIFY.LEFT_ALT);
}
setWatch(btnPressed, BTN, { edge: "rising", repeat: true, debounce: 50 });
```

Nach erfolgreicher Verbindung laesst sich der Code einfach per Klick auf `Save to Flash` in den Speicher des Puck laden. Darauf hin wieder die Verbindung zu Chrome trennen und regulaer den Puck ueber Bluetooth mit Windows 11 verbinden - diese Verbindung wird bei jedem Systemstart erneut hergestellt.

Per Druck auf den Puck - der in sich quasi ein Button ist - schaltet Windows nun HDR an und beim erneuten Druck wieder aus.
